# SSE Editor Plugin

It Provides these plugins

| Plugin    | Description                                                     |
| --------- | --------------------------------------------------------------- |
| Paragraph | text block base tool                                            |
| Header    | header block                                                    |
| Quote     | tool for quotes                                                 |
| Warning   | warning tool can be used as editorials notifications or appeals |
| Delimiter | delimiter tool                                                  |
| Code      | tools for code examples                                         |
